# 👁️ OpticBreak20 — Protect Your Vision

**OpticBreak20** is a modern,Progressive Web App (PWA) designed to help you follow the **20-20-20 rule**. Prevent digital eye strain and keep your vision sharp with a simple, regular habit.

## 🛠️ Tech Stack

*   **HTML5 & CSS3**: Custom modern medical-grade styling (Vanilla CSS).
*   **JavaScript (ES6+)**: Custom timer logic with browser storage integration and Service Worker handling.
*   **Progressive Web App (PWA)**: Built with `manifest.json` and a Custom Service Worker for offline support and native installation.

## ✨ Features

*   **🧘 20-20-20 Reminder**: Tracks your screen time for exactly 20 minutes.
*   **🧘 Mandatory Rest**: When the timer ends, a mandatory 20-second break countdown begins. You can only dismiss it once your eyes have fully relaxed.
*   **🔊 System Alerts**: Even if you're in another tab or the app is minimized, the system's toast notification and alarm ensure you never miss a break.

## 🛠️ Installation & Setup 
If you're running this project locally for development:

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/siddhali24/OpticBreak20.git
    cd eye-break
    ```

2.  **Run with Python**:
    ```bash
    python -m http.server 8000
    ```

3.  **Access in Browser**:
    Open [http://localhost:8000](http://localhost:8000)
